# CommonPopup_SOL
UI组件工具类

>提供Android常用的自定义组件和动画

## 使用方式
### Step 1. Add the JitPack repository to your build file
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
### Step 2. Add the dependency
```
dependencies {
        implementation 'com.github.SeedsOfLove:UIComponentUtils_SOL:1.0.0'
	}
```
